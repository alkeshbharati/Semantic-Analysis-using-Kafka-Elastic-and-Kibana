Steps to run the code:

1. We are using Kafka streaming,elastic search and kibana for visualisation using python    
   in pycharm.

2. Start zookeeper, elastic search and kibana in terminal if you are using Mac.

3. Run StreamProducer.py and then run StreamConsumer.py using pycharm and go to browser  
   http://localhost:5601 to start using kibana for visualization

4. For sentiment analysis I am using Textblob's sentiment polarity in StreamConsumer code
